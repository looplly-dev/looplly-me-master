# Getting Support

Need help? We're here to assist you with any questions or issues you may have.

## Before Contacting Support

Try these quick fixes first:

### Common Issues

**Can't log in?**
- Check your email and password are correct
- Use "Forgot Password" to reset
- Clear browser cache and cookies
- Try a different browser

**Survey not loading?**
- Refresh the page
- Check internet connection
- Update your browser
- Try on a different device

**Missing payment?**
- Check Wallet → Transaction History
- Allow 3-5 business days for processing
- Verify your payment details are correct

**Verification code not received?**
- Wait 2-3 minutes for delivery
- Check spam/junk folder
- Verify mobile number is correct
- Request a new code

## Contact Support

If you still need help, reach out to our team:

### Email Support

**Email:** help@looplly.com

**Response Time:** Within 24 hours (usually faster)

**What to include:**
- Your account email
- Detailed description of the issue
- Screenshots (if applicable)
- Steps you've already tried
- Device and browser information

### Live Chat

**Available:** 9 AM - 5 PM SAST (Monday - Friday)

**Access:** Click the chat icon in your dashboard

**Best for:** Quick questions and immediate assistance

### Support Portal

**URL:** support.looplly.com

## Report Issues

### Technical Problems

Found a bug or technical issue?

1. Go to **Settings** → **Report Issue**
2. Describe the problem
3. Include steps to reproduce
4. Attach screenshots if helpful
5. Submit report

Our team will investigate and follow up.

### Survey Issues

Problems with a specific survey?

**Email:** surveys@looplly.com

**Include:**
- Survey ID (if available)
- Date/time attempted
- Description of problem
- Screenshots of error

### Account Security

Suspicious activity or security concerns?

**Email:** security@looplly.com

**Urgent:** Available 24/7 for security issues

**If compromised:**
1. Reset your password immediately
2. Email security@looplly.com
3. Verify your profile information
4. Enable two-factor authentication

## Escalation Process

Not satisfied with the support response?

1. **Initial Response** - Support agent assists
2. **Supervisor Review** - Request escalation if needed
3. **Management Review** - For complex issues
4. **Final Appeal** - Executive team review

Request escalation by replying to your support ticket with "Escalate to Supervisor".

## Tips for Faster Support

1. **Be Specific** - Detailed descriptions help us help you faster
2. **Include Screenshots** - Visual proof speeds resolution
3. **Check Status First** - Visit support.looplly.com/status for known issues
4. **One Issue Per Ticket** - Separate issues = faster resolution
5. **Follow Up Politely** - We're working through all requests
