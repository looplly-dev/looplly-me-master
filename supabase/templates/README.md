## Template Variables

Supabase provides these variables that are automatically replaced:

- `{{ .Email }}` - User's email address
- `{{ .ConfirmationURL }}` - Verification link (auto-generated by Supabase)
- `{{ .Token }}` - Verification token (if needed separately)
- `{{ .TokenHash }}` - Hashed token (if needed separately)
- `{{ .SiteURL }}` - Your site URL (from Supabase settings)

## Customization

To further customize the template:

1. **Change Colors**: Update the gradient colors in the `<style>` section
   - Primary: `#f97316` (orange)
   - Secondary: `#ea580c` (darker orange)

2. **Update Logo**: Replace the `<div class="logo">L</div>` with an `<img>` tag pointing to your logo

3. **Modify Content**: Edit the welcome message and instructions in the content section

4. **Add Social Links**: Uncomment and update social media links in the footer

## Testing

To test your email templates locally:

1. Run Supabase locally: `supabase start`
2. Access Mailpit at: http://127.0.0.1:54324
3. Register a test user
4. View the email in Mailpit

**Note:** Local Supabase auto-confirms users, so you might not see the email. To test emails:
- Disable auto-confirmation in `config.toml`
- Or test directly in production with a test email
